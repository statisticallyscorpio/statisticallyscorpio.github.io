# ♏ Statistically Scorpio

**Can astrology survive peer review?**

Statistically Scorpio is a crowdsourced data science project exploring whether astrology placements, birth timing, and natal chart features show measurable relationships with real-world outcomes.

The project combines anonymous survey data, statistical analysis, and machine learning to investigate questions such as:

* Are certain placements associated with ADHD, autism, anxiety, or depression?
* Do relationship outcomes vary across chart configurations?
* Are some life experiences more common among particular placements?
* Does astrology provide predictive value beyond demographics and known risk factors?
* Are observed patterns driven by astrology itself, birth timing, or self-perception?

The goal is not to prove or disprove astrology.

The goal is to let the data speak.

## 📊 Methodology

This project follows a data-first approach.

Birth information is used to calculate natal chart features including:

* Sun sign
* Moon sign
* Rising sign
* Planetary placements
* Element distributions
* Modalities
* House placements (when available)

Statistical analyses will compare chart features against reported outcomes while controlling for known confounding variables such as age, education, income, childhood environment, and astrology familiarity.

Machine learning models will be used to evaluate whether chart-derived features provide meaningful predictive value.

---

## ⚠️ Important Notes

This project is exploratory and for educational purposes.

* Participation is anonymous.
* No identifying information is collected.
* Results do not constitute medical, psychological, legal, or financial advice.
* Correlation does not imply causation.

Any findings should be interpreted cautiously and independently replicated before drawing conclusions.
