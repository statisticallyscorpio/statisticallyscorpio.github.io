For years I've been told:

- Scorpios are red flags
- Geminis can't be trusted
- Virgos need to relax
- Mercury is responsible for basically everything

As a data scientist, I have questions.

This project uses anonymous survey responses, statistics, and machine learning to investigate whether astrology placements actually correlate with real-world outcomes—or whether we've all been aggressively pattern-matching for centuries.
